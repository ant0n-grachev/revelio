Revelio Privacy Policy Revelio is a security-focused Chrome extension that detects potentially hidden text on webpages. What data Revelio accesses: Website content on the active tab, only when the user initiates a scan. How data is used: Data is processed locally in the browser to generate scan results and highlights. Data sharing and storage: Revelio does not sell, transfer, or share user data with third parties. Revelio does not transmit scanned page content to external servers. Revelio does not use data for advertising, profiling, creditworthiness, or lending. Results are only copied to clipboard if the user clicks "Copy JSON."
